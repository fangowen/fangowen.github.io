# Personal Portfolio

A static portfolio page for projects, experiments, and selected work.

## Edit Your Projects

Update the `projects` array in `script.js`. Each project supports:

- `title`
- `type` (`software`, `design`, `data`, or `research`)
- `year`
- `description`
- `tags`
- `url`
- `color`

## Edit Your Experience

Update the `experience` array in `script.js`. Each entry supports:

- `role`
- `organization`
- `period`
- `location`
- `summary`
- `highlights`

## Preview

Open `index.html` directly in a browser. No build step is required.

Contact links are filled in from the resume in `index.html`.
